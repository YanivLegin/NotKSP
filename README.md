# 📋 NotKSP - חנות מסחר אלקטרוני לאלקטרוניקה וחשמל

פרויקט זה הוא חנות מסחר אלקטרוני מודרנית ונקייה המציגה מוצרי אלקטרוניקה איכותיים ללא העומס הויזואלי המאפיין את הרשתות הגדולות בישראל.

האתר נבנה כ-Single Page Application (SPA) המבוסס על **React** ו-**Tailwind CSS** ונטען ישירות מהדפדפן ללא צורך בשלבי בנייה (Build steps), מה שמאפשר אירוח מהיר וקל ב-**GitHub Pages**.

## 🚀 פיצ'רים עיקריים

- **סינון לפי קטגוריות:** 8 קטגוריות שונות (מחשבים, טלוויזיה ואודיו, בית חכם, אודיו ואביזרים, לביש, גיימינג, חשמל קטן למטבח, אביזרי מחשב).
- **חיפוש חופשי:** תיבת חיפוש מהירה המסננת מוצרים בזמן אמת לפי שם ותיאור.
- **פרטי מוצר מלאים:** תצוגה מורחבת (Modal) הכוללת מפרט טכני מלא והמלצות על מוצרים דומים מאותה קטגוריה.
- **סל קניות אינטראקטיבי:** ניהול עגלת קניות השמור בדפדפן (`localStorage`) - הוספה, הסרה, ועדכון כמויות.
- **תהליך Checkout מדומה:** מילוי פרטי משלוח וקבלת אישור הזמנה עם מספר הזמנה ייחודי.
- **עיצוב פרימיום:** צבעים מותאמים (כחול פלדה עמוק, ירוק מנטה חשמלי ל-CTA), טיפוגרפיה איכותית (Heebo & Assistant בגוגל פונטס), ואנימציות עדינות לחוויית משתמש מרשימה.
- **רספונסיביות מלאה:** מותאם לחלוטין למובייל וטאבלטים עם תמיכה מלאה בכתיבה מימין לשמאל (RTL).
